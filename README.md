# Lovable Clone

An AI-powered application builder that generates frontend code from natural language prompts using LLM agents and Spring AI.

## Features

* AI-driven frontend code generation from user prompts
* Distributed microservices architecture using Apache Kafka and Saga orchestration
* Secure authentication and authorization with Spring Security and JWT
* Stripe subscription and payment integration
* MinIO object storage for project assets
* PostgreSQL + pgvector for persistence and vector search
* Dockerized deployment for scalable infrastructure

## Tech Stack

* Java 21
* Spring Boot
* Spring AI
* Apache Kafka
* PostgreSQL
* pgvector
* Spring Security
* JWT
* Stripe
* MinIO
* Docker
